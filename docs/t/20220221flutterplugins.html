使ってよかった Flutter プラグイン

Update: 2022-02-21

私は基本的に「寄らば大樹の陰」なので、だいたい Flutter Favorite を使っています。

flex_color_scheme

過去記事: Flutter の色の設定で楽をする

intl

過去記事: Flutter の多言語化で JSON を使いたくない

flutter_native_splash

過去記事: Flutter アプリの起動時にロゴなど表示する

go_router

生の Navigator 2.0 の解説を見て泣きそうになりましたが、これならだいじょうぶです。

equatable

flutter_bloc の state にするクラスの定義で使います。

flutter_markdown

Flutter は文書の掲示には向かないのですが、それでも例えば Privacy policy くらいはアプリの中で表示したいことがありますから。リンクのジャンプには url_launcher を使いました。

shared_preferences

Web では LocalStorage 、 Android では、、、とそれぞれについて同等のプログラムを自分で書きたくはないですから。
