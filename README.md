# michinobumaeda.github.io

Prerequisites

- `rbenv` or something that can handle `.ruby-version`

Test

```
$ git clone git@github.com:MichinobuMaeda/MichinobuMaeda.github.io.git
$ cd MichinobuMaeda.github.io
$ bundle install
$ bundle exec jekyll serve
```

Generate indexes

```
$ ruby mkindex.rb 
```
