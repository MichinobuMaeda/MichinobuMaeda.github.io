# My site

URL: <https://pages.michinobu.jp>

## Development

### Prerequisites

- `nvm` or Node.js >= 22

### Initialize

```bash
git clone git@github.com:MichinobuMaeda/MichinobuMaeda.github.io.git
cd MichinobuMaeda.github.io
npm install
```

### Build and test locally

```bash
npm run build && npm run start
```
